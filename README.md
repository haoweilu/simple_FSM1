# 有限狀態機 (FSM) 設計 - Verilog 實作

本專案實作了一個簡單的**三段式有限狀態機 (Three-segment FSM)**。該狀態機根據輸入訊號 `in` 在四個狀態之間轉換，並在每個狀態輸出對應的 4-bit 資料。

## ⚙️ 系統架構與功能

### 1. 埠口說明 (Ports)
| 訊號名稱 | 類型 | 位元寬度 | 說明 |
| :--- | :--- | :---: | :--- |
| `clk` | Input | 1 | 系統時脈 (Clock)，正邊緣觸發 |
| `rst_n` | Input | 1 | 非同步非同步重設訊號 (Reset)，低電位有效 |
| `in` | Input | 1 | 控制狀態轉換的輸入條件 |
| `out_data` | Output | 4 | 根據目前狀態輸出的 4-bit 資料 (組合邏輯輸出) |

---

### 2. 狀態定義與輸出對照表
本設計包含 4 個狀態，使用 `parameter` 定義：
* `s0` (0)
* `s1` (1)
* `s2` (2)
* `s3` (3)

每個狀態對應的 `out_data` 輸出如下：
| 目前狀態 (State) | 輸出資料 (`out_data`) |
| :---: | :---: |
| `s0` | `4'h0` (0000) |
| `s1` | `4'h1` (0001) |
| `s2` | `4'h8` (1000) |
| `s3` | `4'hf` (1111) |

---

### 3. 狀態轉換邏輯 (State Transition)
狀態間的轉換完全由輸入訊號 `in` 決定：

```text
       in=0          in=1          in=0          in=1
   ┌───┐         ┌───┐         ┌───┐         ┌───┐
   │   │         │   │         │   │         │   │
 ┌─┴───▼─┐ in=1  ▼───┴─┐ in=0  ▼───┴─┐ in=1  ▼───┴─┐
 │  s0   ├──────►│  s1   ├──────►│  s2   ├──────►│  s3   │
 └─▲─────┘       └─────▲─┘       └─────▲─┘       └─┬─────┘
   │                   │               │           │
   └───────────────────┴───────────────┴───────────┘
                         in=0 (s3 轉回 s0)
